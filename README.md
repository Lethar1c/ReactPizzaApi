# React Pizza
Данный проект - учебный, в рамках изучения возможностей ASP.NET Core. Упор делается на backend, хоть и нельзя забывать про frontend.
## Демонстрационная версия проекта
Демо-версия проекта (на 10.02.2025 лишь frontend) доступна по ссылке https://lethar1c.github.io/ReactPizza/
## RoadMap проекта (приблизительная, возможно, будет дополняться, последнее обновление - 10.02.2025)
1. Создание рабочего CRUD пицц на сайте
2. Реализация аутенфикации с использованием JWT и авторизации по ролям (пользователь, администратор, суперадминистратор)
  - Пользователь - может лишь просматривать пиццы и оформлять заказы
  - Администратор - может создавать и редактировать пиццы на сайте, просматривать заказы пользователей
  - Суперадминистратор - может назначать администраторов
3. Реализация личного кабинета пользователя, просмотра его заказов, прошлых заказов, возможность отмены заказа
